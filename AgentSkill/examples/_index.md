# examples（端到端演练包）索引

> 注意：演练包仅用于展示工件格式与证据链，不是默认方案；实际任务必须以本 skill 的 `SKILL.md` 与对应阶段的 `stages/*/SKILL.md` 为准。示例内容可改写、扩展或替换。
>
> 目的：把“阶段模块”串成可直观看懂的闭环：State → Plan/Task → Execute/Write → Validate(证据) → Review(打分与返工)。

每个演练包都包含：
- State（节选）
- Plan（节选）
- Task（节选）
- Review 输出示例（总分 + P0/P1/P2 + 结论）
- Evidence Index 示例（证据入口）
